TITLE.JS

Browser Title Bar Manipulation JavaScript Library With No Dependency.

Download

Compress, Production ( 1.2kb )Uncompress, Developement ( 1.4kb )

Usage

Add Prefix

Example:

<script>
	Title.pref("(2)");
</script>

Click To See On Title BarRefresh

Add Suffix

Example:

<script>
	Title.suf("(2)");
</script>

Click To See On Title BarRefresh

Change Predefined Title

Example:

<script>
	Title.change("Changed Title");
</script>

Click To See On Title BarRefresh

Animation: Marquee Effect

Example:

<script>
	Title.animation("marquee");
</script>

Click To See On Title BarRefresh

Animation: TypeWriter Effect

Example:

<script>
	Title.animation("typeWriter");
</script>

Click To See On Title BarRefresh
